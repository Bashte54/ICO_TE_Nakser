Concepto de Ingeniería de Software

La Ingeniería de Software es la aplicación de un enfoque sistemático, disciplinado y cuantificable al desarrollo, operación y mantenimiento del software, así como el estudio de estos enfoques, es decir, la aplicación de la ingeniería al software.

Definición según IEEE

"La aplicación de un enfoque sistemático, disciplinado y cuantificable al desarrollo, operación y mantenimiento del software; es decir, la aplicación de la ingeniería al software."

Objetivos principales

  • Mejorar la calidad de los productos de software
  • Aumentar la productividad y el trabajo de los ingenieros de software
  • Facilitar el control del proceso de desarrollo de software
  • Suministrar las bases para desarrollar software de manera profesional
  • Proporcionar soporte para el mantenimiento del software

Ciclo de Vida del Desarrollo de Software

El Ciclo de Vida del Desarrollo de Software (SDLC) es un marco que describe las fases para construir y mantener sistemas de software. Proporciona un enfoque sistemático y organizado para el desarrollo de software.
